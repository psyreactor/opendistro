Opendistro is a Ruby wrapper for the Opendistro API this gem only supports Opendistro 1.9.0.

## Installation

Install it from rubygems:

```sh
gem install opendistro
```

Or add to a Gemfile:

```ruby
gem 'opendistro'
# gem 'opendistro', github: 'psyreactor/opendistro'
```
